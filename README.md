# 法人AI素案

Next.js 14で構築された、法人向けAIエージェントシステムです。

## 🎨 デザインコンセプト

**白黒ベースのミニマルデザイン**
- シンプルで洗練されたUI
- 黒と白を基調としたモノクロームパレット
- 直感的で使いやすいインターフェース
- スムーズなアニメーションとマイクロインタラクション

## ✨ 主な機能

### 1. 📊 ダッシュボード
- リアルタイムで進行状況を一目で確認
- タスク完了率、稼働中のAI数、組織効率など
- 進行中の指示と進捗バー
- AIエージェントの状態表示

### 2. 💬 AIチャット
- 経営AIとのリアルタイム対話
- **AIエージェント自動実行フロー**
  - 指示を入力すると自動的にタスクを分解
  - 各部門AIが連携して処理を実行
  - リアルタイムでエージェント間の情報共有を可視化
  - 処理完了まで完全に自動化
- よく使われる質問のクイックアクション
- チャット内で処理状況を確認

### 3. ✅ タスク管理
- **カンバンビュー**: ドラッグ&ドロップ可能なタスクボード
- **リストビュー**: テーブル形式での一覧表示
- タスク作成・編集モーダル
- 優先度、進捗、担当者の管理
- ステータス（Todo / 進行中 / レビュー / 完了）

### 4. 📚 ナレッジベース
- 組織の知識を蓄積・共有
- カテゴリ別の分類
- 評価とタグ付け
- 使用回数のトラッキング

### 5. 📄 ドキュメント管理
- レポート、提案書、マニュアル、契約書の管理
- タイプ別フィルタリング
- 部門別の分類
- 公開/下書きのステータス管理

### 6. 🏢 組織管理
- **組織図・マインドマップ表示**
- 経営AI → 部門AI → 統合AIの3層構造
- 情報フローの可視化
- 各AIエージェントの稼働状況

### 7. 🔔 通知
- システムからのリアルタイム通知
- 未読/既読の管理
- タイプ別アイコン表示

### 8. ⚙️ 設定
- プロフィール管理
- 通知設定
- AI設定

## 🚀 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **アイコン**: Heroicons
- **状態管理**: Zustand
- **日付処理**: date-fns
- **データ可視化**: Recharts

## 📦 セットアップ

### インストール

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

### ビルド

```bash
npm run build
```

### 本番環境起動

```bash
npm start
```

## 🎯 デザインの特徴

### カラーパレット
- **ベース**: 白 (#FFFFFF)
- **テキスト**: 黒 (#000000)
- **グレー**: 複数のグレーシェード
- **アクセント**: 黒（ボタン、アクティブ状態）

### UIコンポーネント
- **カード**: 白背景、グレーボーダー、シャドウ
- **ボタン**: 黒背景、白テキスト
- **入力フィールド**: グレーボーダー、フォーカス時に黒リング
- **アクティブ状態**: 黒背景、白テキスト

# 法人AIエージェントシステム

Next.js 14で構築された、法人向けAIエージェント統合システムです。

## 🚀 クイックスタート

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## ✨ 主な機能

- 📊 **ダッシュボード** - 全体の状況を一目で確認
- 📈 **アナリティクス** - ビジネスメトリクスの可視化
- 💬 **AIチャット** - Claude AIとナレッジベースを活用した対話
- 🎯 **ゴール管理** - プロジェクト目標の設定と追跡
- 🤖 **AIエージェント** - 複数エージェントの連携実行
- 📚 **ナレッジベース** - 組織知識の蓄積と検索
- 📄 **ドキュメント管理** - ファイルのアップロードと管理
- 🏢 **組織図** - 組織構造の可視化
- 🔔 **通知** - リアルタイム通知
- ⚙️ **設定** - プロフィールとシステム設定

## 🛠️ 技術スタック

- **フロントエンド**: Next.js 14, React, TypeScript, Tailwind CSS
- **バックエンド**: Next.js API Routes, Supabase
- **AI**: Claude API (Anthropic)
- **データベース**: PostgreSQL (Supabase)
- **認証**: Supabase Auth

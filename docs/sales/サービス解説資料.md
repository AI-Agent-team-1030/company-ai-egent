# 法人自走型AIエージェント サービス解説資料

---

## サービスコンセプト

### 経営の意思が、組織全体に届く。

AIエージェントが実現する、次世代の組織運営プラットフォームです。

---

## 企業が抱える3つの課題

### 1. 意思伝達の遅延

経営方針が現場に届くまでに時間がかかり、部門間で解釈がブレてしまう。結果として、スピード感のある意思決定ができなくなっています。

### 2. 属人化とナレッジ分散

重要な判断が特定の社員に依存し、過去の成功事例や失敗が蓄積されていない。担当者の異動や退職により、貴重なノウハウが消失してしまいます。

### 3. 部門間の連携コスト

部門を横断するプロジェクトで調整に時間を取られ、情報の重複入力や確認作業が頻繁に発生しています。

---

## 本サービスが提供する価値

単なるAIチャットツールではありません。

**社内のナレッジをAIが学習し、誰でも・いつでも・最適な情報にアクセスできる環境**を構築します。

経営者が戦略を語れば、AIがそれを解釈し、社員一人ひとりが必要な情報を即座に取得できる。これにより、組織全体の意思決定スピードと品質が向上します。

---

## 主要機能

### 機能1：AIナレッジアシスタント

**できること**

- 自然な言葉で質問するだけで、社内ドキュメントから最適な回答を自動生成
- 複数のAIモデル（Claude / GPT / Gemini）から最適なものを選択可能
- 回答の根拠となる引用元を自動表示（ソース別に色分け）

**得られる価値**

- 検索時間の大幅削減
- 新人の即戦力化
- 24時間対応

---

### 機能2：ナレッジベース管理

**できること**

- PDF、Word、Excel、PowerPoint、画像など、あらゆる形式のドキュメントに対応
- AIが自動でテキストを抽出し、検索可能な状態に変換
- 画像内の文字もOCRで自動認識
- Gemini File Searchによる高精度なセマンティック検索

**得られる価値**

- ナレッジの一元化
- 暗黙知の形式知化
- 情報の陳腐化防止

---

### 機能3：Google Drive連携

**できること**

- Google Workspaceのドライブと直接連携
- ドライブ内のファイルを自動でAI検索対象に
- Google Docs/Sheets/Slidesも自動変換して検索可能
- 会社レベルで接続、全社員が利用可能
- 差分同期機能で効率的な更新

**得られる価値**

- 既存資産の活用
- 二重管理の解消
- 導入コストの削減

---

### 機能4：マルチAI対応

**できること**

- 用途に応じて最適なAIモデルを選択可能
- Anthropic Claude、OpenAI GPT、Google Gemini に対応
- Geminiは標準搭載（追加APIキー不要）

**得られる価値**

- コスト最適化
- ベンダーロックイン回避
- 最新技術の活用

## 技術基盤

| 項目 | 技術 |
|------|------|
| フレームワーク | Next.js 14（App Router） |
| フロントエンド | React 18 + TypeScript + Tailwind CSS |

| ホスティング | Google Cloud Run |

| 暗号化 | AES-256-CBC（APIキー保護） |

## セキュリティ

- お客様のデータは暗号化され、安全に管理
- データは他社と共有されることは一切なし
- お客様専用の環境で完全に独立して運用
- APIキーはAES-256暗号化で保存
- 通信はHTTPS（TLS 1.3）で保護
- Firestoreセキュリティルールによる企業別データ分離

# バージョン管理

このドキュメントは、法人AIエージェントシステムの各バージョンの機能と進化の記録です。

---

## 📦 Version 1.0 - MVP版（2025年11月24日）

**コンセプト**: 最小限の社内チャットボットシステム

### ✨ 搭載機能

#### 1. AIチャット 💬
- 社内のことを質問したら答えてくれる
- ナレッジベースを参照した回答生成（RAG）
- 会話履歴の保存と検索
- フォルダー別のチャット整理
- Claude APIによるリアルタイム対話

#### 2. ナレッジベース 📚
- 組織の知識を蓄積・共有
- フォルダー階層での整理
- ドキュメントのアップロード（PDF, Word, Excel, CSV, 画像）
- 全文検索機能
- AIチャットでの参照（RAG統合）
- 社員がナレッジを追加・編集可能

#### 3. 設定 ⚙️
- プロフィール管理
- ユーザー名の設定
- Claude APIキーの管理
- 通知設定

#### 4. 認証 🔐
- メール/パスワード認証
- セッション管理
- ログイン/ログアウト

### 🎯 ターゲット

- 社内情報を効率的に共有したい企業
- 社員が簡単にナレッジを蓄積・検索できる環境を作りたい組織
- AIを活用した社内Q&Aシステムを導入したい企業

### 💡 使い方

1. 社員がナレッジベースに情報を登録
2. 他の社員がAIチャットで質問
3. AIがナレッジベースを参照して回答
4. 必要に応じて新しいナレッジを追加

## 🚀 Version 2.0 - エンタープライズ版（予定）

**コンセプト**: 組織全体のパフォーマンス可視化と管理

### 追加機能

#### 1. ホーム（ダッシュボード + アナリティクス統合）📊
- 組織の健康状態を一目で確認
- 主要指標（タスク完了率、稼働AI数、組織効率、KPI達成率など）
- 部門別パフォーマンスとAI稼働率のグラフ
- AIエージェントの活動状況
- 詳細なインサイトと分析結果
- 具体的な提案・アクションプラン

#### 2. ドキュメント管理 📄
- レポート、提案書、マニュアル、契約書の管理
- タイプ別フィルタリング
- 部門別の分類
- バージョン管理
- 公開/下書きのステータス管理

#### 3. 組織図 🏢
- 組織構造の可視化
- 部門とメンバーの管理
- 役割と権限の設定

#### 4. 通知 🔔
- システムからのリアルタイム通知
- 未読/既読の管理
- タイプ別アイコン表示

### 復元方法
- `_archive/v2_features/` フォルダ内のページを `app/(app)/` に移動
- `docs/archive/VERSION_BACKUP.md` の手順に従う

## 🎯 Version 3.0 - AI自動化版（予定）

**コンセプト**: AIエージェントによる業務自動化

### 追加機能

#### 1. タスク管理（ゴール + タスク統合）📋
- ゴール作成（「いつまでにこれやって」と入力）
- エージェントが自動でタスクを分解
- タスクリストの自動生成
- リアルタイム進捗確認
- 成果物の自動生成（会議資料、レポート、提案書）

#### 2. AIエージェント管理 🤖
- 複数のAIエージェントの管理
- エージェント間の自動連携
- タスクの自動分配と実行
- エージェントの稼働状況モニタリング
- カスタムエージェントの作成

### 復元方法
- `_archive/v3_features/` フォルダ内のページを復元
- `docs/archive/VERSION_BACKUP.md` の手順に従う

## 📝 バージョンアップの方針

### Version 1.0 → 2.0
- 組織のパフォーマンス可視化が必要になった時
- 部門別の管理が必要になった時
- データドリブンな意思決定を行いたい時

### Version 2.0 → 3.0
- 業務の自動化を本格的に進めたい時
- AIエージェントによるタスク実行が必要になった時
- 大規模な組織での運用が必要になった時

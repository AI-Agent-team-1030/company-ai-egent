# 法人AI ロードマップ - MVP版

**現在のバージョン**: Version 1.0 - MVP版
**最終更新**: 2025年11月24日
**戦略**: 最小限の機能で社内ナレッジ共有とAIチャットボットを実現

---

## 🎯 Version 1.0 - MVP版の目的

### コンセプト
**社内ナレッジ共有 × AIチャットボット**

社員がナレッジを登録し、AIが社内情報に基づいて質問に答える、シンプルで使いやすいシステム。

### ターゲット
- 社内情報を効率的に共有したい企業
- 社員が簡単にナレッジを蓄積・検索できる環境を作りたい組織
- AIを活用した社内Q&Aシステムを導入したい企業

---

## ✨ MVP版の機能（3つ）

### 1. AIチャット 💬 **社内チャットボット**

**機能**:
- 社内のことを質問したら答えてくれる
- ナレッジベースを参照した回答生成（RAG）
- 会話履歴の保存と検索
- フォルダー別のチャット整理
- Claude APIによるリアルタイム対話

**使い方**:
```
社員: 「経費精算の手順を教えてください」
AI: 「ナレッジベースから関連情報を検索中...
     経費精算は以下の手順で行います：
     1. 経費精算システムにログイン
     2. ...（ナレッジベースの情報を参照して回答）」
```

**価値**:
- すぐに社内情報を見つけられる
- 他の社員に聞く手間が省ける
- 24時間いつでも質問できる

---

### 2. ナレッジベース 📚

**機能**:
- 組織の知識を蓄積・共有
- フォルダー階層での整理
- ドキュメントのアップロード（PDF, Word, Excel, CSV, 画像）
- 全文検索機能
- AIチャットでの参照（RAG統合）

**使い方**:
```
1. 管理者や社員がナレッジを登録
   - マニュアル、手順書、Q&Aなど

2. フォルダーで整理
   - 営業部門/契約手順
   - 総務部門/経費精算
   - IT部門/システム利用方法

3. AIチャットが自動で参照
   - 社員が質問すると自動で検索
```

**価値**:
- 組織の知識が蓄積される
- 新入社員もすぐに情報を見つけられる
- 属人化を防げる

---

### 3. 設定 ⚙️

**機能**:
- プロフィール管理
- ユーザー名の設定
- Claude APIキーの管理
- 通知設定

**価値**:
- ユーザーごとにカスタマイズ可能
- セキュアなAPIキー管理

## 🎯 Version 3.0 - AI自動化版（予定）

**コンセプト**: AIエージェントによる業務自動化

### 追加機能

#### 1. タスク管理（ゴール + タスク統合）📋
- ゴール作成（「いつまでにこれやって」と入力）
- エージェントが自動でタスクを分解
- タスクリストの自動生成
- リアルタイム進捗確認
- 成果物の自動生成（会議資料、レポート、提案書）

**使い方**:
```
ユーザー: 「来週までに新規事業の提案書を作成してください」
  ↓
エージェントが自動で以下のタスクに分解:
1. 市場調査
2. 競合分析
3. 提案書のドラフト作成
4. レビューと修正
  ↓
各タスクを自動実行
  ↓
提案書が完成
```

#### 2. AIエージェント管理 🤖
- 複数のAIエージェントの管理
- エージェント間の自動連携
- タスクの自動分配と実行
- エージェントの稼働状況モニタリング
- カスタムエージェントの作成

## 💡 MVP版の改善計画

### 1. AIチャット改善
- **RAG最適化**（最優先）
  - 全ナレッジではなく、関連性の高いものだけを参照
  - 回答速度の向上

- **参照元の明示**（超重要）
  - どのナレッジを参照したか表示
  - 出典を明確にする

- **ストリーミング応答の改善**
  - より自然な対話体験

### 2. ナレッジベース改善
- **チャットからナレッジ化**
  - よく聞かれる質問を簡単にナレッジ化
  - ボタン1つで保存

- **検索の高速化**
  - より高速な全文検索

- **部署内共有 or 全社共有の選択**
  - アクセス権限の管理

### 3. マルチテナント対応（重要）
- 法人単位でのデータ分離
- 部署単位でのナレッジ共有
- 詳細は `docs/MULTI_TENANT_DESIGN.md` を参照
